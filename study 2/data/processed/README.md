# Processed Data Directory

This directory is reserved for processed datasets generated by scripts in `code/analysis/`.

## Notes

- Files here should be reproducible from the raw data in `data/raw/`.
- To regenerate processed outputs, run the analysis pipeline:
  - `source("code/analysis/run_hlm_analysis.R")`

## Workflow Summary

1. Read raw Weibo feature files (`data/raw/weibo/`).
2. Read user profile file (`data/raw/AllUidV5.txt`).
3. Read city-level files (`data/raw/city_level/`).
4. Clean, merge, filter, and standardize data.
5. Export model-ready and result datasets.

## Version Control Guidance

- Processed files may be large and are usually better excluded from version control.
- For reproducibility, keep raw data and analysis scripts as the source of truth.
