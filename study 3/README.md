# Study 3 Data and Analysis

This directory provides district-level data, analysis scripts, and result files for Study 3 of the manuscript, facilitating direct access to research inputs, processed data, and statistical outputs.

## Directory Structure

- `data/raw/`: Original input data for Study 3 (district-level macro data, population density, coordinates)
- `data/processed/`: Analysis data after sample construction (N=816) and standardized versions
- `code/reproduce/`: Reproducible analysis scripts
- `docs/`: Variable definitions, workflow documentation, and manuscript references
- `results/`: Correlation analysis, mediation analysis, and SEM results corresponding to manuscript tables

## How to Run

### Requirements

Python 3.10+ is recommended. Install dependencies in the `study3` directory:

```bash
pip install -r requirements.txt
```

### Main Analysis Script (Table 6 & 7)

Run in the `study3` directory:

```bash
python code/reproduce/run_study3_analysis.py
```

Optional parameters:

```bash
python code/reproduce/run_study3_analysis.py --boot 5000 --seed 8911
```

### SEM Model (Table 8 & 9)

To run the SEM model:

```bash
python code/reproduce/run_table9_sem_from_original.py
```

## Output Files

### Generated by run_study3_analysis.py:
- `results/table6_correlations.csv` - Correlation matrix (with significance markers)
- `results/table7_bootstrap_mediation.csv` - Bootstrap mediation analysis results

### Generated by run_table9_sem_from_original.py:
- `results/table9_sem_paths_from_program_table9_style.csv` - SEM path coefficients
- `results/table9_sem_fit_from_program.csv` - SEM fit indices


## Notes

- The analysis sample is constructed following Study 3's original filtering logic, resulting in 816 district units.
- Table 6 and Table 7 are computed directly from data by `run_study3_analysis.py`.
- Table 8 and Table 9 SEM models require running `run_table9_sem_from_original.py` separately.
